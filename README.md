# Architecting AI Agents: Building, Evaluating, and Shipping Agentic AI Systems

A practical guide to designing, building, evaluating, and operating agentic AI system. The book also includes a full interview-prep companion for agentic AI system interviews.

**Full book:**
* [Architecting AI Agents (PDF)](architecting-ai-agents.pdf)
* [Architecting AI Agents (Markdown)](architecting-ai-agents.md)

## Repo layout

- `chapter_NN_*.md` — individual chapters, the source of truth for all content.
- `architecting-ai-agents.md` / `architecting-ai-agents.pdf` — the assembled full book, generated automatically from the chapter files.
- `book.css` — stylesheet used when rendering the PDF.
- `.github/workflows/` — CI that concatenates the chapters and rebuilds the Markdown/PDF whenever a `chapter_*.md` file changes on `main`.

## Contributing

Edit the individual `chapter_NN_*.md` files, not `architecting-ai-agents.md` or the PDF — both are build artifacts regenerated by GitHub Actions on every push to `main` that touches a chapter file.
